HOW TO:

To use this calculator, make sure to identify the given values in the problem correctly. Analyze what is the initial, current, and future population, if given. And as well as the initial, current, and final time.

To use this app, select FROM THE OPTIONS, based on what you need and what is unknown in the problem. Next, INPUT the given values in the appropriate input box and choose the unit of your value. Last, CLICK SOLVE and CHECK THE SOLUTION BELOW FOR THE ANSWERS.

Also, if you want to answer again, you can either 1) Re-enter the value and click solve if you have another problem with the same unknown, or 2) You can click GO BACK and select from the OPTIONS to solve again for different problems.
